# ESP32 Project

Цей проєкт є навчальним. 

## Основні особливості

* **📦 Модульність**: Вся специфічна логіка роботи з датчиком винесена в окремий компонент `sensor`, що дозволяє легко перевикористовувати код.
* **⚙️ Гнучка конфігурація (Kconfig)**: Параметри проєкту (наприклад, активація датчика або вибір пінів) винесені в графічне меню `menuconfig`.
* **🤖 Автоматизація (sdkconfig.defaults)**: Проєкт містить файл базових налаштувань, який автоматично встановлює частоту процесора 240 МГц та інші системні параметри при першій збірці.
* **📝 Документованість**: Код поки що не містить ніяких коментарів.

## 🛠️ Вимоги

Щоб запустити цей проєкт, вам знадобиться:
1.  **Плата**: ESP32-S3-DEVKITC-1 (або будь-яка інша плата на базі чипа ESP32).
2.  **Середовище**: Встановлений ESP-IDF Framework v5.5.2.
3.  **Редактор**: VS Code з розширенням "ESP-IDF Extension".

## 🚀 Як зібрати та запустити

Кроки для встановлення та запуску проекта:

1.  **Клонування репозиторію**:
    ```bash
    git clone git@github.com:shemliyivan/esp32_repo.git
    cd esp32_repo
    ```

2.  **Встановлення цільового чипа**:
    ```bash
    idf.py set-target esp32s3
    ```

3.  **Налаштування параметрів**:
    Ви можете змінити параметри компонента `Sensor` у меню:
    ```bash
    idf.py menuconfig
    ```

4.  **Збірка, прошивка та запуск монітора**:
    ```bash
    idf.py build flash monitor
    ```

## 📂 Структура проєкту

* **`main/`** — точка входу в програму (`main.c`).
* **`sensor/`** — кастомний компонент, поки нічого не містить.
    * `Kconfig` — опис меню налаштувань компонента.
    * `CMakeLists.txt` — правила збірки компонента.
* **`sdkconfig.defaults`** — файл, що містить базові налаштування проєкту.
* **`.vscode/`** — конфігураційні файли для IntelliSense та дебагу в VS Code.
