# ESP32 Project

Цей проєкт є навчальним. 

## 🛠️ Вимоги

Щоб запустити цей проєкт, вам знадобиться:
1.  **Плата**: ESP32-S3-DEVKITC-1 (або будь-яка інша плата на базі чипа ESP32).
2.  **Середовище**: Встановлений ESP-IDF Framework v5.5.2.
3.  **Редактор**: VS Code з розширенням "ESP-IDF Extension".

## 🚀 Як зібрати та запустити

Кроки для встановлення та запуску проекта:

1.  **Клонування репозиторію**:
    ```bash
    git clone git@github.com:shemliyivan/esp32_repo.git
    cd esp32_repo
    ```

2.  **Встановлення цільового чипа**:
    ```bash
    idf.py set-target esp32s3
    ```

3.  **Налаштування параметрів**:
    Ви можете змінити параметри компонентів у меню:
    ```bash
    idf.py menuconfig
    ```

4.  **Збірка, прошивка та запуск монітора**:
    ```bash
    idf.py build flash monitor
    ```

## Основні можливості

* Багатоканальна телеметрія: збір та обробка даних з сенсорів LSM6DS3 (орієнтація), AHT20 (вологість та температура) та BMP280 (атмосферний тиск).
* Керування приводами: синхронна робота з кроковим двигуном через драйвер ULN2003 та сервоприводом.
* Інтерфейс керування: підтримка аналогового джойстика для ручного маніпулювання та світлодіодна індикація режимів роботи.
* Дистанційний зв'язок: передача структурованих даних через MQTT та підтримка локального з'єднання Bluetooth Low Energy (BLE).

## Технологічний стек

* Платформа: мікроконтролер ESP32-S3.
* Фреймворк: ESP-IDF.
* Операційна система: FreeRTOS для керування пріоритетами та задачами.
* Периферійні інтерфейси: використання шин I2C та SPI для комунікації з сенсорами, UART для налагодження системи.
* Мережеві протоколи: MQTT для хмарної телеметрії та BLE для мобільного доступу.
* Формат даних: серіалізація повідомлень у форматі JSON.

## Структура проєкту

Нижче наведено ієрархію основних компонентів системи:

```text
.
├── components/
│   ├── aht20_driver/       # Драйвер сенсора температури та вологості
│   ├── bluetooth_component/# Налаштування стека BLE
│   ├── bmp280_driver/      # Драйвер сенсора атмосферного тиску
│   ├── i2c_bus/            # Конфігурація та менеджер шини I2C
│   ├── lsm6ds3_driver/     # Драйвер акселерометра та гіроскопа
│   ├── mqtt_component/     # Клієнт для роботи з MQTT брокером
│   ├── my_joystick/        # Обробка сигналів з джойстика
│   ├── my_led/             # Логіка керування світлодіодними індикаторами
│   ├── my_servo/           # Керування кутом повороту сервопривода
│   ├── my_wifi/            # Менеджер підключення до Wi-Fi мережі
│   ├── spi_bus/            # Конфігурація та менеджер шини SPI
│   ├── stepper/            # Керування кроками двигуна ULN2003
│   └── uart_communicator/  # Налагодження та логування через UART
├── main/
│   ├── main.c              # Основна точка входу та ініціалізація задач
│   └── CMakeLists.txt
├── managed_components/     # Бібліотеки Espressif (button, led_strip, mqtt, servo)
└── sdkconfig.defaults      # Оптимізовані системні налаштування
```
